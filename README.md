# Teste - Backend Lemon - Elegibilidade

## Estrutura do Projeto

```
lemon-challenge-be/
├── src/
│   ├── common/
│   │   ├── mensagens.js
│   │   ├── round.js
│   │   ├── tipos.js
│   ├── services/
│   │   ├── eligibility.js
│   │   ├── validateInput.js
│   ├── index.js
├── tests/
│   ├── eligipility.test.js
│   ├── validateInput.test.js
├── package.json
```

## Como Executar

1. Instale as dependências:
    ```sh
    npm install
    ```

2. Inicie o servidor:
    ```sh
    npm start
    ```

3. Execute os testes:
    ```sh
    npm test
    ```

---

Este projeto foi implementado em `Node.js` utilizando o framework `Express`.
